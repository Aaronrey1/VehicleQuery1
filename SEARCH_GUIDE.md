# VehicleDB Pro - Search Guide

## Individual Vehicle Search

Use the **Search** tab to find vehicles by make, model, year, device type, or port type.

### Steps to Search:

1. **Go to Search Tab**
   - Click on "Search" in the navigation bar
   - You'll see the vehicle search form

2. **Select Search Criteria** (optional - you can use any combination)
   - **Make**: Choose a vehicle manufacturer (e.g., Toyota, Honda, Ford)
   - **Model**: Select a specific model (filters based on selected make)
   - **Year**: Pick a manufacturing year
   - **Device Type**: Filter by compatible device type
   - **Port Type**: Filter by connection port type

3. **Click "Search Vehicles"**
   - Results appear below the search form
   - Shows 10 vehicles per page by default

### Search Results Features:

- **Sorting**: Click column headers (Make, Model, Year) to sort results
- **Pagination**: Use "Previous" and "Next" buttons to browse pages
- **Export**: Click "Export" to download results as CSV file
- **Total Count**: See how many vehicles match your search

### Search Tips:

- Leave filters empty to view all vehicles
- Use Make + Model for specific vehicle lines
- Combine Device Type or Port Type to find compatible devices
- Sort results by clicking column headers

## Bulk Search

Use the **Bulk Search** tab to search for multiple vehicles at once.

### Steps for Bulk Search:

1. **Go to Bulk Search Tab**
   - Click on "Bulk Search" in the navigation bar

2. **Add Vehicle Queries**
   - Click "+ Add Vehicle" button
   - Each query has three fields:
     - **Make**: Vehicle manufacturer
     - **Model**: Vehicle model name
     - **Year**: Manufacturing year

3. **Fill in Multiple Vehicles**
   - Add as many vehicles as you need
   - Example queries:
     - Toyota Camry 2020
     - Honda Accord 2019
     - Ford F-150 2021

4. **Remove Unwanted Queries**
   - Click the "Remove" button (trash icon) next to any query you want to delete

5. **Click "Search All Vehicles"**
   - All matching vehicles appear in the results table below

### Bulk Search Results:

- **Combined Results**: All matching vehicles from all queries shown together
- **Device & Port Info**: See device types and port types for each match
- **Export All**: Download all bulk search results as CSV
- **Total Count**: Shows total vehicles found across all queries

### Bulk Search Tips:

- Perfect for checking multiple vehicles at once
- Useful for fleet compatibility checks
- Export results to share with others
- Add up to as many vehicles as you need

## Understanding Results

### Vehicle Information Displayed:

- **Make**: Vehicle manufacturer
- **Model**: Vehicle model name
- **Year**: Manufacturing year
- **Device Type**: Compatible device model number
- **Port Type**: Connection port specification

### Export Feature:

Both individual and bulk search results can be exported:
1. Click the "Export" button above results
2. CSV file downloads automatically
3. Open in Excel, Google Sheets, or any spreadsheet program
4. Contains all search results (not just current page)
